# Prompt Logic Analysis 🧠🤖

[![GitHub release](https://img.shields.io/github/v/release/NDT29thDec/prompt-logic-analysis?style=flat-square)](https://github.com/NDT29thDec/prompt-logic-analysis/releases)

Một kho lưu trữ chuyên dụng để phân tích, thiết kế và tối ưu hóa cấu trúc logic của các prompt dành cho Trí tuệ Nhân tạo (AI - LLMs).

## 📖 Giới thiệu (About)

Trong kỷ nguyên của các Mô hình ngôn ngữ lớn (Large Language Models), việc cung cấp cho AI một câu lệnh (prompt) đơn thuần là chưa đủ để giải quyết các tác vụ phức tạp. Dự án **Prompt Logic Analysis** được tạo ra với mục tiêu:
1. Mổ xẻ và phân tích cách AI tiếp nhận các tầng logic trong prompt.
2. Xây dựng các luồng tư duy chặt chẽ (như *Chain of Thought*, *Tree of Thoughts*).
3. Đảm bảo AI đưa ra các câu trả lời chính xác, đáng tin cậy và không bị ảo giác (hallucination).

## 📂 Cấu trúc dự án (Repository Structure)

- `logic-analysis.txt`: File core chứa các phân tích chi tiết, bộ quy tắc logic, và các mẫu prompt đã được thử nghiệm thực tế.
- `README.md`: Tài liệu hướng dẫn tổng quan về dự án.
- *(Các thư mục/files sẽ được cập nhật thêm trong tương lai)*

## ✨ Tính năng & Nội dung chính (Key Contents)

- **Phân tích tư duy logic:** Cách chia nhỏ vấn đề (breakdown) để AI có thể hiểu và lập luận từng bước.
- **Khắc phục lỗi logic của AI:** Nhận diện các điểm mù (blind spots) của LLMs và cách đặt điều kiện (constraints) vào prompt để hạn chế sai sót.
- **Khuôn mẫu Prompt (Prompt Templates):** Các đoạn text mẫu đã được tinh chỉnh logic tối đa để bạn có thể sao chép và sử dụng ngay cho công việc, học tập hoặc lập trình.

## 🚀 Cách sử dụng (How to Use)

1. Mở file `logic-analysis.txt` để đọc các nguyên tắc phân tích logic.
2. Sao chép các cấu trúc prompt phù hợp với bài toán của bạn.
3. Tích hợp trực tiếp vào các nền tảng AI như ChatGPT, Claude, Gemini hoặc tích hợp vào API của ứng dụng bạn đang phát triển.
4. Tùy chỉnh các biến số (variables) trong prompt để ra được kết quả như ý.
